## How to capture them

**From the live Netlify site (recommended — includes the animated background, fonts, everything):**
1. Open each page in your browser at a reasonably wide window (~1440px works well)
2. Mac: `Cmd+Shift+4`, then drag to select the browser window content. Windows: `Win+Shift+S`.
3. For a cleaner crop, use your browser's dev tools → "Capture full size screenshot" (Chrome: open DevTools → Cmd/Ctrl+Shift+P → type "screenshot")

**Recommended size:** roughly 1440×900 or your actual viewport size — the README displays them at full width, so they'll scale down automatically on GitHub.

Save each file directly in this folder with the exact filenames above, then commit and push — the broken image icons in the main README will resolve automatically once the files exist.
